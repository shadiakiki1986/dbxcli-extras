## dbxcli-sync

dbxcli sync command that solves https://github.com/dropbox/dbxcli/issues/53


## Install

```
pip3 install git+ssh://git@github.com/shadiakiki1986/dbxcli-sync.git@v0.0.1
```

## Usage

```
dbxcli_sync [--verbose={0,1,2}] <localdir> <dbxdir>
```


## Dev notes

```
pip3 install pew
pew new dbxcli_sync
```
